HOW TO USE

on Windows
1. Open Command Prompt 
2. Navigate to directory where it's installed
	ie. (cd downloads)
3. Type the following
	MultiplicationTable "A NUMBER OF YOUR CHOICE"
4. The program will create a workbook of this digit you created
